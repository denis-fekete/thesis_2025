\chapter{Dataset Creator installation}
\label{datasetCreatorInstall}
\textbf{Dataset Creator} was packaged as a Python package. Installation requires: \textbf{Python 3.10} and \textbf{pip} installed. Installation is done with the command line interface in a few steps:

\subsubsection{Download the Git repository and move into it}
\begin{lstlisting}
git clone https://github.com/denis-fekete/cvlibg-dataset-creator.git
cd cvlibg-dataset-creator
\end{lstlisting}

\subsubsection{Create and activate the virtual environment}
\begin{lstlisting}
python -m venv venv
# Linux:
    source venv/bin/activate
# or Windows:
    .\venv\Scripts\activate
\end{lstlisting}

\subsubsection{Install dependencies}
\begin{lstlisting}
pip install .
\end{lstlisting}
or for development:
\begin{lstlisting}
pip install -e .
\end{lstlisting}

\subsubsection{Install \texttt{torch} and \texttt{torchvision} with GPU support}
By default, non-GPU-accelerated versions of \texttt{torch} and \texttt{torchvision} are installed. To enable this feature, it is necessary to uninstall old versions and install new versions based on the machine's hardware. For more, see \url{https://pytorch.org/get-started/locally/}.
\begin{lstlisting}[breaklines=true]
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu129
\end{lstlisting}

\subsubsection{Open the application}
\begin{verbatim}
dataset-creator
\end{verbatim}

\chapter{Installing mobile library with Android projects}
\label{mobileInstall}
Computer Vision Library For Board Games is compiled with \textbf{JVM target 11} and \texttt{Kotlin 2.0.21}. Projects should use Kotlin version \textbf{2.0.x} or newer compatible versions.

To include the library in the Android project, it is necessary to add the Jitpack repository. This is done inside the \textbf{settings.gradle.kts} file, in the project's root folder:
\begin{verbatim}
    dependencyResolutionManagement {
        repositories {
            mavenCentral()
            maven { url = uri("https://jitpack.io") }
        }
    }
\end{verbatim}

Next, add the library dependency to the module's \textbf{build.gradle.kts} configuration file. 
\begin{verbatim}
    dependencies {
        implementation("com.github.denis-fekete:cvlibg:0.9.8")
        implementation("com.github.denis-fekete:cvlibg-yolo-onnx:0.9.8")
    }
\end{verbatim}

The \textbf{0.9.8} tag should be replaced with the desired version, which can be found at: \url{https://jitpack.io/#denis-fekete/computer_vision_library_for_board_games}
